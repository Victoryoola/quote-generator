# Simple Random Quote Generator
This is a simple random quote generator created with REACT.
It displays quotes and allows users to generate a new quote by clicking the refresh button.

## Features
1. Displays random quotes from a list
2. Clicking the refresh button changes quotes
3. React Hook ("useState")

# Tech used
1. React - Frontend framework
2. CSS - Styling
3. Font awesome - animated refresh icon

## Quick start

Requirements: Node.js (>=16) and npm.

1. Install dependencies
   - Open PowerShell in the project root:
     npm install

2. Start dev server:
     npm start

3. Open http://localhost:3000
